weixin4j-qy
===========

[微信企业号](http://qydev.weixin.qq.com/wiki/index.php)开发工具包
---------------------------------------------------------------

功能列表
-------
* **weixin4j-qy-api**

  + DepartApi `部门管理API`
	
  + UserApi `成员管理API`
  
  + TagApi `标签管理API`
  
  + MediaApi `多媒体管理API`
  
  + MenuApi `菜单管理API`
  
  + NotifyApi `消息发送API`

* **weixin4j-qy-server**

  + `netty服务器` & `消息分发`
  
  + 回调连接`AES`加密、解密
